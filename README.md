# 🏭 Function Factory

An interactive web-based learning tool designed to introduce students to the concept of **mathematical functions** through exploration and problem-solving.

## 🌟 Overview
The game presents a "Black Box" factory. Students send numbers ($x$) into the machine and observe the results ($y$). By analyzing the relationship between input and output, they must deduce the underlying algebraic rule.

## 🎮 How to Play
1. **Choose a Level:** Select between Primary, Junior High, or High School difficulty.
2. **Test Inputs:** Use the top keypad to send numbers into the machine.
3. **Observe Patterns:** Check the **History Log** to see how the numbers change.
4. **Solve:** Use the specialized math keypad to type the function (e.g., `f(x) = 2x + 3`) and verify your answer.

## 🛠 Features
- **Dynamic Difficulty:** Functions scale from simple addition to quadratic equations ($x^2$).
- **Integrated Math Keypad:** Custom buttons for $x$, $·$, and $x^2$ ensure students focus on the math, not keyboard syntax.
- **Visual Feedback:** Animated "processing" phase helps students visualize the transformation.
- **Fully Responsive:** Works perfectly on Chromebooks, iPads, and mobile devices.

## 🚀 Installation
1. Clone this repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.
3. (Optional) Host it for free using **GitHub Pages**.

## 🏫 Educational Value
This tool supports the transition from arithmetic to algebraic thinking by emphasizing the concept of a function as a consistent "rule" applied to a variable.
