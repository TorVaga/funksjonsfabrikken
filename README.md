# Funksjonsfabrikken 🏭

Et interaktivt matematikkspill for elever i alle aldre. 

## Om spillet
Elevene velger et siffer som sendes inn i "fabrikken". Ut kommer et resultat basert på en hemmelig matematisk funksjon. Målet er å gjette funksjonen ved å observere sammenhengen mellom input ($x$) og output ($y$).

### Nivåer:
- **Nivå 1**: Enkel addisjon og subtraksjon (Barneskole).
- **Nivå 2**: Lineære funksjoner som $ax + b$ (Ungdomsskole).
- **Nivå 3**: Potensfunksjoner som $x^2 + n$ (Videregående).

Laget av [TorVaga](https://github.com/TorVaga/).
